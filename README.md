# Cesar Castro - Interactive CV/Resume

A modern, interactive CV website showcasing 10+ years of full-stack engineering experience, technical leadership, and architecture expertise.

## Features

✨ **Interactive Design**

- **Expandable Experience Cards** - Click any job to expand and view full details
- **Skill Filtering** - Filter experience by clicking any technology skill
- **Sticky Navigation** - Quick access to different CV sections
- **Smooth Animations** - Fade-in effects and progressive element loading
- **Responsive Layout** - Works beautifully on desktop, tablet, and mobile

## Tech Stack

**Frontend:**

## Getting Started

### Prerequisites

- Node.js 16+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone <repo-url>
cd ce-castro.github.io

# Install dependencies
npm install

# Start development server
npm run dev
```

The site will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

Output will be in the `dist/` directory.

### Preview Production Build

```bash
npm run preview
```

## Usage

- **Explore Experience**: Click on any job title to expand and view achievements and responsibilities
- **Filter by Skills**: Click any skill badge to filter experience by that technology
- **Clear Filters**: Click the ✕ button on the filter badge or re-click a skill to reset
- **Navigate**: Use the sticky navigation bar for quick access to different sections

## Key Sections

- **Header** - Professional introduction and quick snapshot
- **Highlights** - Key accomplishments and expertise areas
- **Professional Summary** - Overview of technical and leadership capabilities
- **Experience** - Detailed work history with interactive cards
- **Skills** - Organized by category (Frontend, Backend, Data & APIs, DevOps, Leadership)
- **Contact** - Get in touch section with multiple contact methods
